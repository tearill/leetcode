# 相同的树  
- leetcode👉https://leetcode-cn.com/problems/same-tree/

给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。  

- 思路  
  递归  

- 三部曲  
  1. 递归的终止条件：返回值(判断条件)为 false 的时候  
  2. 找返回值：true/false  
  3. 本级递归应该做什么：判断左右子树是否相同  

- 判断左右子树相等  
  排除初始情况 --- 两棵树都为空返回 true
  排除一些不相等的情况之后就是相等的情况  
  1. 左右子树一个为空，一个非空  
  2. 左右子树结点的值不一致
