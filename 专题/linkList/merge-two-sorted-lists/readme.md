# 合并两个有序链表  
leetcode👉https://leetcode-cn.com/problems/merge-two-sorted-lists/  

将两个升序链表合并为一个新的**升序**链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的  

示例：
```js
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

- 分析  
  排列两个有序的链表  
  如果 l1 的头节点 < l2 的头节点，拼接 l1 的头节点，否则拼接 l2 的头节点  

- 递归  
  退出条件为两个链表中有一个为空  
  每层递归判断链表节点值的大小，拼接值较小的那个  
  返回值为头结点  
- 迭代  
  与递归思路相同  
- 使用数组  
  将两个链表的值放进一个数组进行排序  
  把排好序的数组变成一个新的链表返回
