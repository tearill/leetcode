# Pow(x, n)  
leetcode👉https://leetcode-cn.com/problems/powx-n/  

实现 pow(x, n) ，即计算 x 的 n 次幂函数。  
```js
示例 1:
输入: 2.00000, 10
输出: 1024.00000

示例 2:
输入: 2.10000, 3
输出: 9.26100

示例 3:
输入: 2.00000, -2
输出: 0.25000
解释: 2^-2 = 1/2^2 = 1/4 = 0.25
```

- 取巧  
  Math.pow()  

- 暴力法  
  取 n 的绝对值，先计算 result = x ^ n => 也就是 |n| 个 x 相乘  
  1. 如果 n > 0 直接返回 result  
  2. 如果 n < 0 返回 1/result  

- 递归  
  x^n = x^(2/n) * x^(2/n) * x^(n%2)
